* Research Status
** Emerging Research Area
Recent studies (2024-2025) have begun investigating how *code structure*
and *syntax formatting* affect LLM code generation quality, providing
theoretical foundation for our practical observations:

*** Directly Supporting Research
- [[https://dblp.org/rec/journals/corr/abs-2604-24712][Prompt Structure Effects on LLM Code Generation (2025)]]:
  Demonstrates that syntactic structure and formatting significantly impact
  code correctness in generated outputs.

- [[https://dblp.org/rec/journals/corr/abs-2604-19826][Test Syntax Structure Affects Foundation Model Code Generation (2025)]]:
  Empirical evidence that code structural properties influence generation quality.
  *Closest existing work to our practical observation.*

- [[https://dblp.org/rec/journals/corr/abs-2412-15305][Tree-of-Code: Tree-Structured Code Generation (2024)]]:
  Utilizes hierarchical tree structures (analogous to parenthesis nesting)
  for improved generation.

** Specific Gap for Lisp
While general code structure effects are now being studied,
*dangling vs. compact parenthesis styles* remain unexamined in academic literature.
No published studies specifically address:
- Parenthesis layout as a formatting variable for AI-assisted coding
- Lisp-specific structural aids for generation quality
- Comparative error rates by formatting style
